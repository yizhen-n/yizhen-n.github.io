Cando

2017
UI
Animation
This is an UI exercise of a desktop todo application. Through simulating the office desk in real-world, I aimed to make simple interface for user to focus on their tasks, from "to do" to "can do".

UI Ideation

Icons: Associating with objects on the desk, and crafting to the simplest figure.
Colors: For people sitting long, use eye-protective greens as primary colors, and a few gradients to decorate the plain interface.

Animation concepts

1. Accomplishment
When user checks the task, the ticket disappears with a spring-curve animation to convey the happiness of accomplishment. I also used an animation to emphasize the 'Done' category on left navigation bar, so user can easily find checked tickets there in case of wrong operation.


2. Add a task
The new ticket move in as a post-it on the desk. During editing, the pure background helps user focus on adding information. When finish, an animation on left navigation bar is used to emphasize where the ticket goes.


3. Focus on the task
Hover is a good type of interaction on desktop, as it can save the effort of click. When user hovers on the left half of each ticket, a certain operations on this ticket can be done, including focus mode. The focus mode is like a lamp on your desk. You put all the attention on one task and control the pace via countdown feature.



All the animations above were made in Framer.
